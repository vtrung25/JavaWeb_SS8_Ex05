
Danh sách Annotation theo từng quy tắc nghiệp vụ:

| Quy tắc nghiệp vụ                  | Loại Annotation        | Annotation cụ thể                  | Ghi chú |
|------------------------------------|------------------------|------------------------------------|---------|
| Mã Tour (tourCode)                 | Custom Field-Level     | `@TourCode`                        | Regex kiểm tra VN_/INT_ + 5 chữ số |
| Giá adultPrice                     | Annotation chuẩn       | `@Positive`                        | > 0 |
| Giá childPrice                     | Annotation chuẩn + Custom Class-Level | `@Positive` + `@ValidPriceRange` | childPrice ≤ adultPrice |
| startDate                          | Annotation chuẩn       | `@FutureOrPresent`                 | ≥ ngày hôm nay |
| endDate                            | Custom Class-Level     | `@ValidDateRange`                  | endDate > startDate (cross-field) |
